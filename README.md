# 🐯 BaDang
BaDang(바당)은 AI를 활용한 시니어 자영업자 대상 온라인 가게 정보 분석 서비스입니다.
이 저장소는 BaDang의 프론트엔드 개발을 담당하는 레포지토리입니다.
